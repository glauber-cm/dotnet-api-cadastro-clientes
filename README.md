# API de Cadastro de Clientes

API REST desenvolvida em ASP.NET Core para gerenciamento de clientes.

## 🚀 Tecnologias
- .NET 8
- ASP.NET Core
- Entity Framework Core
- SQL Server

## 📌 Funcionalidades
- Criar cliente
- Listar clientes
- Buscar por ID
- Atualizar cliente
- Remover cliente

## 🧱 Arquitetura
- Controllers
- Services
- Repositories
- DTOs
